Overview

A Spira::Type for integer values. Values will be associated with the XSD.integer type.

A Resource property can reference this type as Spira::Types::Integer, Integer, or XSD.integer.

def self.serialize(value)
  RDF::Literal.new(value.is_a?(BigDecimal) ? value.to_s('F') : value.to_s, datatype: RDF::XSD.decimal)
end

def self.unserialize(value)
  object = value.object
  object.is_a?(BigDecimal) ? object : BigDecimal.new(object.to_s)
end
